# CSS Advanced Project

## Project Overview
This project is part of the ALX Frontend curriculum and focuses on **CSS and styling**. The goal is to recreate a web page design provided in a Figma file using **only HTML, CSS, and minimal JavaScript**. External libraries such as Bootstrap, React, or Vue are **not allowed**.

The project emphasizes:
- Applying CSS properties and selectors effectively.
- Structuring semantic HTML.
- Matching a design mockup from Figma.
- Understanding CSS specificity, box model, and layout techniques.

---

## Learning Objectives
By completing this project, you will be able to:

1. Explain what CSS is and how it is applied to HTML elements.
2. Use classes and selectors to style elements efficiently.
3. Compute CSS specificity and apply it to style elements correctly.
4. Apply box model properties: margin, padding, borders, width, and height.
5. Use layout techniques such as Flexbox and CSS Grid.
6. Understand how browsers load and render a webpage.
7. Create a fully functional, W3C-compliant web page that closely resembles the Figma design.

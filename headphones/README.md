# Headphones Landing Page

A modern, responsive landing page for a headphones product, built with HTML and CSS. This project is part of the **ALX HTML & CSS program** and focuses on creating a clean, professional, and mobile-friendly layout.

## Project Overview

This landing page demonstrates:

- A responsive design that adapts to desktop and mobile screens (≤480px)
- A hero/header section with a call-to-action
- Sections for "What we do...", "Our results", and "Contact Us"
- A custom footer
- Animated elements for better engagement
- A hamburger menu for mobile navigation

![Landing Page Preview](./ce72a25e-c7ad-4636-a1dd-38c53748a530.png)

---

## Features

- **Responsive Layout**: Fully adapts to mobile devices, tablets, and desktops.
- **Custom Fonts**: Uses `Source Sans Pro` and `Spin Cycle OT`.
- **Reusable Components**: Icons and pentagons are built using CSS for easy reusability.
- **Smooth Animations**: Interactive hover effects for buttons, icons, and sections.
- **Accessibility Considerations**: Clear labels, form fields, and focus states.

Notes & Best Practices
Fonts: Install Source Sans Pro and Spin Cycle OT for proper rendering.

CSS Structure:

Reset CSS styles

Use variables for colors, fonts, and spacing

Generic selectors for maintainability

Avoid overly specific selectors

Design Approach: Start building pages from outside to inside, top to bottom.

Responsive Design: Max content width 1000px, centered, with breakpoints for ≤480px.

Interactions:

Links hover/active: #FF6565

Button hover/active: opacity: 0.9
